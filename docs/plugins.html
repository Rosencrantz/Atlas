Dropdowns

About

The dropdown plugin defines a very simple pattern. At it's most basic the dropdown is simply a trigger with an an associated container that is displayed or hidden based on user interation. The dropdown is positioned relative to its trigger, usually aligned to the bottom left corner. The contents of the dropdown can be anything from a menu to a form.

Markup

<a id="trigger" data-trigger="dropdown" aria-owns="container">Dropdown</a>
<div id="container">
	<ul>
		<li><a href="http://www.google.com">Google</a></li>
		<ll><a href="http://www.yahoo.com">Yahoo</a></li>
		<li><a href="http://www.bing.com">Bing</a></li>
	</ul>
</div>

JavaScript

$('#trigger').dropdown()

Methods

$('#trigger').dropdown('show');

$('#trigger').dropdown('hide');

$('#trigger').dropdown('toggle');
