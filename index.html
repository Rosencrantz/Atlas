Atlas Introduction

Welcome to Atlas, A small tool for quickly and easily adding common UI components into your application

Dropdowns

Dropdowns, as the name suggests are containers that drop down below a trigger. A dropdown can contain pretty much anything, although it's most popular function is as a container for a menu.

Examples

Dropdown 1 Dropdown 2

Markup

<a data-trigger="dropdown" aria-owns="mydropdown">Trigger</a>
<div id="mydropdown">Container</div>

Options

Unsurprisingly a dropdown will tend to drop down, but with a little configuration these dropdowns can be setup to appear in a multitude of locations.

<a data-trigger="dropdown" aria-owns="mydropdown" data-align="right" data-valign="top" >Trigger</a>
<div id="mydropdown">Container</div>

Horizontal alignment can be one of five values: farLeft, left, center, right or farRight. Vertical navigation can also be one of five seperate values: above, top, middle, bottom, below

Javascript invocation

Creating and interacting with dropdowns from your javacsript is possible with the following simple commands

//Converts the #selector to a dropdown. The selector should have the appropriate aria-owns value
$('#selector').dropdown();

//Will show, hide or toggle the dropdown
$('#selector').dropdown('show');
$('#selector').dropdown('hide');
$('#selector').dropdown('toggle');

Events

A dropdown will generate a number of pre/post events for you to hook into for altering and extending the functionality of your dropdowns. The events raised by the dropdown are:

  • atlas.dropdown.show (raised immediately before a dropdown is displayed
  • atlas.dropdown.shown (raised immediately after a dropdown is displayed)
  • atlas.dropdown.hide (raised immediately before a dropdown is hidden)
  • atlas.dropdown.hidden (raised immediately after a dropdown is hidden)
